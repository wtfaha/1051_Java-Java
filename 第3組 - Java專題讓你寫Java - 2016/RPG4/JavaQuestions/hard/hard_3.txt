Given:

public class Hello {
    String title;
    int value;

    public Hello() {
        title += " World";
    }

    public Hello(int value) {
        this.value = value;
        title = "Hello";
        Hello();
    }
} 

And:
Hello c = new Hello(5);
System.out.print(c.title);
What is the result?
	A. Hello
	B. Hello World
	C. Compilation fails.
	D. Hello World 5
